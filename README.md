# King-s-Soap

## Description

King's Soap is a local, family-owned business founded by Anita King in 2016. Anita King is self-taught and has
devoted many hours to refining her soapmaking process and creating a brand that she is passionate and proud of.
She offers a variety of options including organic and all-natural selections, avoiding harsh chemicals in every bar.​

The client needs a fully functioning website for their company, King’s Soap. The site will feature the company's products, allow purchases, and manage inventory, incorporating payment and shipping APIs.Visitors and admins will have unique login information that is used to sign in and sign out of their accounts. Customers will be able to browse the company's products, make purchases, and view order history, while the admin can access additional features such as editing, updating, and deleting product information.

Check back in with us in December when we plan to launch the new website!​

### Installing utilizing VS Code

* How/where to download your program

#### 1: Install Node.js
1. Open PowerShell or Command Prompt
2. Go to where they want the project
Example:
cd C:\
3. Clone the repo
git clone https://github.com/SteveNavarr0/King-s-Soap.git
4. Go into the project
cd King-s-Soap
5. Go into the frontend
cd frontend
6. Install the React app dependencies
npm install

#### 2: Install tailwind css:
1. In terminal
cd frontend
2. Install and connect to vite
npm install -D tailwindcss @tailwindcss/vite

#### 3: Install Supabase and add your API keys:
1. In terminal
cd frontend
2. Install and connect to vite
npm install @supabase/supabase-js 
3. Add API keys to your personal .env folder
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
